# 🧮 PHP Calculator

A simple calculator made with **HTML**, **CSS**, and **PHP**.  
It can do basic math like add, subtract, multiply, and divide. ✨

## ✨ Features

- ➕ Add
- ➖ Subtract
- ✖️ Multiply
- ➗ Divide
- ✅ Empty field check
- ✅ Number validation
- ✅ Divide-by-zero protection
- 🌙 Dark theme
- 📱 Responsive design

## 🛠 Tech Stack

- **PHP** — logic
- **HTML** — structure
- **CSS** — styling

## 🚀 How to Run

1. Install **XAMPP**, **WAMP**, or **MAMP**.
2. Put the project inside your server folder.
3. Start **Apache**.
4. Open this link in your browser:

```bash
http://localhost/calculator-php/index.php
```

## 📁 Folder Structure

```bash
calculator-php/
├── css/
│   ├── reset.css
│   └── main.css
├── index.php
└── README.md
```

## 🔍 How It Works

- Enter two numbers.
- Choose an operator.
- Click submit.
- PHP checks the input and shows the result.

## 💡 Notes

- Works with basic arithmetic only.
- Best used with a local server like XAMPP or WAMP.
